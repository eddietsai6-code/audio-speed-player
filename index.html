Audio Speed Player

A tiny reusable browser plugin for local audio practice, tempo changes, and drop-in lesson pages.

Drop-in usage No dependencies 
<script type="module" src="./dist/audio-speed-player.js"></script>
<audio-speed-player></audio-speed-player>
Local first Audio files stay in the browser when a student or teacher chooses a local file.
Pitch aware The player keeps pitch enabled by default while changing playback speed.
Reusable Use the custom element in static HTML now, then publish the same module to npm later.
